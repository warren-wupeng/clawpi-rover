# ClawPi Rover：基于树莓派的视觉语音四轮机器人

**ClawPi Rover** 是一个开源的智能机器人项目，它基于树莓派 5 打造，集成了 **OpenClaw** 作为 AI 大脑，支持视觉识别、语音对话和全向移动。该项目旨在为创客、开发者和爱好者提供一个高度可扩展的平台，让你可以轻松构建自己的智能机器人，并探索 AI 与硬件结合的无尽可能。

## ✨ 特性

- 🚗 **四轮全向移动**：采用麦克纳姆轮 + 四路独立驱动，实现前后、平移、旋转等任意方向移动。
- 👁️ **视觉感知**：搭载 2 自由度云台摄像头，可实时采集图像，结合 OpenClaw 实现目标检测、场景理解等功能。
- 🎤 **语音交互**：使用 USB 麦克风阵列和扬声器，支持语音唤醒、语音识别（ASR）和语音合成（TTS），实现自然的人机对话。
- 🧠 **AI 大脑**：内置 OpenClaw 作为核心 AI 框架，可本地运行轻量模型或调用云端大模型（如 DeepSeek、智谱等），提供强大的对话与推理能力。
- 🔧 **模块化设计**：硬件分层布局，软件采用微服务架构，便于扩展和替换组件（如加装机械臂、激光雷达等）。
- 🌐 **远程控制**：支持通过 Web 界面或手机 App 远程操控，实时查看摄像头画面。

## 💻 软件架构

ClawPi Rover 的软件分为三层：

1. **底层控制层**（ESP32）：负责读取传感器数据、驱动电机、控制云台舵机，通过串口与树莓派通信，确保实时性。
2. **核心逻辑层**（树莓派 5）：运行 OpenClaw AI 框架，整合视觉、语音、对话等模块。OpenClaw 采用 **网关模式**，可调用云端大模型（如 DeepSeek、智谱 GLM），也支持本地运行轻量模型（需配置加速棒）。
3. **用户交互层**：提供 Web 控制界面（基于 Flask + Socket.IO）和语音交互界面，支持远程监控和指令下发。

### 主要软件模块
- **视觉模块**：基于 OpenCV 和 YOLO 等模型，实现人脸识别、目标检测、场景描述。
- **语音模块**：使用 Vosk（离线 ASR）或云 API 进行语音识别，TTS 采用 Piper 或云服务。
- **AI 对话**：OpenClaw 核心，负责对话管理、工具调用（如控制电机、查询天气等）。
- **运动控制**：根据 OpenClaw 的决策或远程指令，生成运动指令并发送给 ESP32。

## 🚀 快速开始

### 1. 硬件组装
- 按照上述清单采购所有硬件。
- 安装底盘、电机、轮子，固定好树莓派、ESP32、电池等。
- 连接电路：**特别注意**电机驱动需要独立电源，不能从树莓派 GPIO 取电；5V 传感器需通过电平转换与树莓派连接。
- 安装摄像头云台，并连接舵机驱动板。

### 2. 软件安装
#### 树莓派环境配置
- 烧录 Raspberry Pi OS (64位) 到 microSD 卡。
- 连接网络，开启 SSH。
- 安装基础依赖：
  ```bash
  sudo apt update
  sudo apt install python3-pip git
  ```
- 克隆本仓库：
  ```bash
  git clone https://github.com/yourname/clawpi-rover.git
  cd clawpi-rover
  ```
- 安装 Python 依赖：
  ```bash
  pip3 install -r requirements.txt
  ```

#### OpenClaw 安装
参考 [OpenClaw 官方文档](https://openclaw.ai/docs) 进行安装，建议采用一键脚本：
```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

#### ESP32 固件烧录
- 使用 Arduino IDE 或 PlatformIO 打开 `firmware/esp32_motor_control` 项目。
- 配置 WiFi 和树莓派 IP，烧录到 ESP32。

### 3. 配置 OpenClaw
- 编辑 OpenClaw 配置文件 `~/.openclaw/config.yaml`，设置所需插件（如摄像头、语音、Telegram 等）。
- 根据需要选择云端 API 密钥（如智谱 GLM 有免费额度）或本地模型路径。

### 4. 启动机器人
- 确保 ESP32 已上电并与树莓派串口连接。
- 启动 OpenClaw 核心服务：
  ```bash
  openclaw start
  ```
- 启动机器人控制 Web 服务：
  ```bash
  python3 web_control.py
  ```
- 在浏览器中访问 `http://<树莓派IP>:5000` 即可看到控制界面和摄像头画面。

## 📖 使用说明

### 语音交互
对着麦克风说出唤醒词（如"嘿，Claw"），机器人会激活并等待指令。你可以问它问题、让它移动、拍照等。例如：
- "向前走一米"
- "左转"
- "这是什么物体？"（它会通过摄像头识别并回答）
- "今天天气怎么样？"（需联网）

### 远程控制
通过 Web 界面可以手动控制机器人移动、调节云台角度，并实时查看摄像头画面。
